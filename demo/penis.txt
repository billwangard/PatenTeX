% Annotation Counter
% 
% use \alabel{foo} to define an annotation that will be referred to using
%
% \alab{foo}    ... alias to alabel
% \alabdup{foo} ... "duplicates" a label in the drawing ... is actually a reference to the alabel
%
% \flab{foo}  ... underlined alabel or "FIELD" label.  These types of labels are not next to leader lines
%                 in the drawing.  Refer to a surface or region.
